# 💰 Finance Dashboard

A modern, interactive finance dashboard built with React to help users track income, expenses, and spending patterns.

![Dashboard Preview](https://via.placeholder.com/800x400?text=Finance+Dashboard)

## ✨ Features

### Dashboard Overview
- **Summary Cards**: Total Balance, Monthly Income, Monthly Expenses, Total Savings
- **12-Month Revenue Chart**: Interactive bar chart with hover tooltips
- **Expense Breakdown**: View by Category or by Month with progress bars

### Transaction Management
- **Search**: Filter transactions by name or category
- **Filter**: Income only / Expense only
- **Sort**: By date (newest/oldest) or amount (highest/lowest)
- **Date Range**: Filter between specific dates

### Role-Based Access
- **Admin Mode**: Can add new transactions
- **Viewer Mode**: Read-only access
- Toggle between roles instantly

### Financial Insights
- Highest spending category
- Savings rate percentage
- Top income source
- Monthly goal achievement

## 🛠️ Tech Stack

- **React 18** - UI Framework
- **Vite** - Build Tool
- **CSS-in-JS** - Styling
- **React Hooks** - State Management
